# Student Result Analyzer

A beginner Python project that analyzes a student's marks.

## Features

- Takes marks for 5 subjects
- Calculates total marks
- Calculates average marks
- Finds the highest mark
- Finds the lowest mark

## Concepts Used

- Python
- Lists
- For Loops
- If Statements
- User Input
- f-Strings
